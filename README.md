# Conversor de Temperaturas
## Fórmulas
1. Celsius To Fahrenheit -> F = (C x 9/5) + 32

2. Fahrenheit To Celsius -> C = (F - 32) x 5/9

3. Celsius To Kelvin -> K = C + 273.15

4. Kelvin To Celsius -> C = K - 273.15

5. Celsius To Réaumur -> R = C x 4/5

6. Celsius To Rankine -> Ra = (C + 273.15) x 9/5
